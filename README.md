Real-time AR flower effect using OpenCV + MediaPipe that replicates the Instagram filter.
Left hand controls stem growth, right hand controls flower blooming through 3 stages — bud, tulip, and full lotus bloom.
 *Controls
- Left hand — spread fingers open to grow stems
- Right hand — spread fingers open to bloom flowers

*Built With
- Python
- OpenCV
- MediaPipe
- NumPy
